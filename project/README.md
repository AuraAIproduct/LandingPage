# AtlasAI Landing Page

Modern, responsive landing page for AtlasAI - the AI co-pilot for real estate.

## Features

- **Home Page**: Hero section, features, demo request, and founder profiles
- **Videos Page**: Embedded YouTube videos (founder intro + product demo)
- **Pitch Deck Page**: Embedded Google Slides presentation
- **Responsive Design**: Works on all devices
- **Smooth Navigation**: React Router with hash navigation support

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Pages

- `/` - Home page with all main content
- `/videos` - Founder intro and product demo videos
- `/pitch-deck` - Embedded Google Slides presentation
